# revis_ml_contagens_visual_vants
Este código é referente à análise estatística da comparação entre as contagens de pinípedes em monitoramentos de voos de VANTs (drones) comparados com monitoramentos visuais. Além de, se existe diferença entre as contagens de duas espécies (Otaria flavescens e Arctocephalusa australis) em relação aos tipos de monitoramentos.
